# Gleam Pro Cleaning — Commercial Cleaning Website

This repository contains the Next.js website for **Gleam Pro Cleaning**, a
commercial-focused cleaning business serving Metro Vancouver.

---

## Focus

- Commercial cleaning (B2B)
- Restaurants (primary niche)
- Offices and community facilities (secondary)

---

## Key Features

- Commercial-first positioning
- Request a Free Walk-Through conversion flow
- RFQ-style walk-through form
- Email lead delivery via Resend
- Scalable commercial service structure

---

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Resend (email delivery)
- Vercel (hosting)

---

## Development Workflow

- Use conventional commits
- Update `CHANGELOG.md` for every commit
- Keep work incremental and test locally
- Refer to `RESUME.md` as the source of truth

---

## Local Development

```bash
npm install
npm run dev
```
